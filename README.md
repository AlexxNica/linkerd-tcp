# Galadriel is an experiment in building TCP middleware in Rust

Quick howto

## Building and Running

Mac/Linux

* Install Rust with `curl -sSf https://static.rust-lang.org/rustup.sh | sh`
* `cargo build`
* `target/debug/proxy2`

## FAQ

Usage: `proxy2 source_addr:port target_addr:port`
